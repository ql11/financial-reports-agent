# 架构设计文档

## 概述

本项目采用基于Agent的微服务架构，将财报分析系统分解为多个独立的Agent，每个Agent负责特定的功能模块。这种架构提供了更好的模块化、可扩展性和可维护性。

## Agent设计

### 1. 数据提取Agent (`data_extraction_agent`)
- **职责**：从PDF财报文件中提取结构化数据
- **输入**：PDF文件路径
- **输出**：结构化财务数据（JSON格式）
- **技能**：PDF处理技能
- **配置**：`agents/data_extraction/agent.yaml`

### 2. 财务分析Agent (`financial_analysis_agent`)
- **职责**：执行核心财务分析
- **输入**：结构化财务数据
- **输出**：财务指标和分析结果
- **技能**：数据分析技能
- **配置**：`agents/financial_analysis/agent.yaml`

### 3. 风险评估Agent (`risk_assessment_agent`)
- **职责**：识别和评估财务风险
- **输入**：财务指标和分析结果
- **输出**：风险评估报告
- **技能**：风险检测技能
- **配置**：`agents/risk_assessment/agent.yaml`

### 4. 报告生成Agent (`report_generation_agent`)
- **职责**：生成最终分析报告
- **输入**：所有分析结果
- **输出**：格式化报告（Markdown/HTML/PDF）
- **技能**：报告写作技能
- **配置**：`agents/report_generation/agent.yaml`

## 技能设计

### 1. PDF处理技能 (`pdf_processing`)
- **功能**：从PDF提取文本和表格
- **技术**：pdfplumber, camelot-py, PyMuPDF
- **输出**：结构化文本和表格数据
- **配置**：`skills/pdf_processing/skill.yaml`

### 2. 数据分析技能 (`data_analysis`)
- **功能**：计算财务指标，执行统计分析
- **技术**：pandas, numpy, scipy
- **输出**：财务指标和趋势分析
- **配置**：`skills/data_analysis/skill.yaml`

### 3. 风险检测技能 (`risk_detection`)
- **功能**：识别财务风险信号
- **技术**：scikit-learn, 阈值分析
- **输出**：风险评分和建议
- **配置**：`skills/risk_detection/skill.yaml`

### 4. 报告写作技能 (`report_writing`)
- **功能**：生成格式化报告
- **技术**：Jinja2, Markdown, WeasyPrint
- **输出**：多种格式的报告
- **配置**：`skills/report_writing/skill.yaml`

## 工作流设计

### 完整分析工作流
1. **PDF文件验证** → 2. **数据提取** → 3. **数据分析** → 4. **风险评估** → 5. **报告生成** → 6. **质量检查**

### 快速分析工作流
1. **数据提取和分析** → 2. **风险评估** → 3. **报告生成**

## 数据流

```
PDF文件 → 数据提取Agent → 结构化数据 → 财务分析Agent → 财务指标
                                                      ↓
风险评估Agent → 风险报告 → 报告生成Agent → 最终报告
```

## 扩展性设计

### 添加新Agent
1. 在 `agents/` 目录下创建新目录
2. 创建 `agent.yaml` 配置文件
3. 在 `configs/main_config.yaml` 中注册
4. 实现Agent逻辑（可选）

### 添加新技能
1. 在 `skills/` 目录下创建新目录
2. 创建 `skill.yaml` 配置文件
3. 在相关Agent中引用
4. 实现技能逻辑（可选）

### 自定义工作流
1. 在 `workflows/` 目录下创建YAML文件
2. 定义步骤和顺序
3. 配置输入输出映射
4. 设置错误处理
