# 增强版财报造假分析系统

## 📋 项目概述

本项目是一个专业的财报造假分析系统，专门用于深度分析上市公司财务报告，识别财务异常和造假风险。系统基于真实财报数据，采用先进的财务分析算法，能够准确检测财报造假迹象。

**核心能力**：
- 🔍 深度分析上市公司财务报告
- ⚠️ 识别财报造假风险和方法
- 📊 计算和解读关键财务指标
- 📈 提供专业的投资建议
- 🎯 基于真实数据的深度分析

## 🚨 项目背景

本项目最初基于模拟数据进行财报分析，但在分析英洛华2025年报时，发现原系统存在严重缺陷：
- 使用模拟数据，无法发现真实财报中的异常
- 只进行静态比率分析，缺乏趋势和逻辑分析
- 风险评级严重错误（原系统：低风险4.8/10，实际：极高风险）

**改进后的系统**基于真实财报数据，能够准确识别：
1. 业绩背离：收入下降但利润增长，现金流暴跌
2. 资产异常：应收账款和存货异常增长，减值准备反向变动
3. 利润操纵：政府补助暴增，递延收益突击摊销
4. 运营异常：产能利用率下降但仍逆势扩产
5. 历史风险：公司有内控违规前科

## 🎯 核心功能

### 1. 核心业绩背离分析
- **收入与利润背离**：检测收入下降但利润增长的异常
- **现金流与利润背离**：检测经营现金流与净利润的严重不匹配
- **收入与现金流背离**：检测收入与销售现金流的异常关系

### 2. 资产负债表异常检测
- **应收账款异常**：检测应收账款异常增长与坏账准备减少
- **存货异常**：检测存货积压与跌价准备减少
- **资产质量分析**：评估资产真实性和减值风险

### 3. 利润操纵分析
- **非经常性损益分析**：检测政府补助等非经常性损益对利润的影响
- **递延收益分析**：检测递延收益的突击摊销
- **利润质量评估**：评估利润的可持续性和质量

### 4. 运营效率分析
- **产能利用率分析**：检测产能利用率下降但仍扩产的异常
- **产销存分析**：检测库存积压与销售不畅的矛盾
- **投资效率评估**：评估投资决策的合理性

### 5. 公司治理风险分析
- **历史违规记录检查**：检查公司历史违规记录
- **关联交易风险**：评估关联交易披露风险
- **内控风险评估**：评估公司内部控制有效性

### 6. 造假路径识别
- **收入虚增路径**：通过应收账款操纵收入
- **利润调节路径**：通过非经常性损益调节利润
- **资产虚增路径**：少提减值准备虚增资产
- **现金流操纵路径**：利润与现金流严重背离
- **关联交易路径**：通过关联方虚构交易

## 🚀 快速开始

### 1. 环境准备
```bash
# 安装依赖
pip install -r requirements.txt

# 如果需要PDF解析功能，安装额外依赖
pip install pdfplumber
```

### 2. 使用命令行工具
```bash
# 分析单个财报文件
python main.py analyze path/to/report.pdf

# 指定公司名称和报告年度
python main.py analyze path/to/report.pdf --company "英洛华科技" --year 2025

# 批量分析多个财报文件
python main.py batch path/to/reports/*.pdf

# 指定输出目录和格式
python main.py analyze path/to/report.pdf --output custom_reports/ --format json

# 显示详细输出
python main.py analyze path/to/report.pdf --verbose
```

### 3. 使用Python API
```python
from src.core.analyzer import FinancialFraudAnalyzer

# 创建分析器
analyzer = FinancialFraudAnalyzer(output_dir="reports")

# 分析财报
report = analyzer.analyze(
    pdf_path="path/to/report.pdf",
    company_name="英洛华科技",
    report_year=2025
)

# 查看分析结果
print(f"风险等级: {report.risk_assessment.risk_level.value}")
print(f"风险评分: {report.risk_assessment.total_score:.1f}/50")
print(f"投资建议: {report.investment_recommendation}")
```

### 4. 批量分析
```python
from src.core.analyzer import FinancialFraudAnalyzer

# 创建分析器
analyzer = FinancialFraudAnalyzer(output_dir="batch_reports")

# 批量分析
pdf_files = [
    "path/to/report1.pdf",
    "path/to/report2.pdf",
    "path/to/report3.pdf"
]

reports = analyzer.batch_analyze(pdf_files, "batch_reports")
print(f"成功分析 {len(reports)} 个文件")
```

### 3. 自定义分析
将PDF财报文件放入 `source/` 目录，然后修改代码中的文件路径：
```python
pdf_path = "source/你的财报文件.pdf"
```

## 🔍 案例分析：英洛华2025年报

### 原系统分析结果（错误）
- **风险评级**：低风险（4.8/10，B级）
- **检测指标**：2个（关联交易异常、会计政策变更）
- **投资建议**：可考虑投资

### 增强版系统分析结果（正确）
- **风险评级**：极高风险（45/50）
- **检测指标**：17个（14个高风险，3个中风险）
- **投资建议**：强烈建议暂停投资，进行深入尽职调查

### 关键发现
1. **业绩严重背离**：收入下降3.11%，利润增长0.93%，现金流暴跌41.5%
2. **应收账款异常**：增长12.58%，坏账准备减少40万
3. **存货异常**：增长21.33%，跌价准备减少147万
4. **政府补助操纵**：增长115%，递延收益摊销增长26倍
5. **产能利用率下降**：固定资产周转率持续下滑，但在建工程增长54%
6. **历史违规前科**：2021年被浙江证监局警示
