# 源文件夹说明

此文件夹用于存放待分析的PDF格式财务报告文件。

## 使用方法

1. 将PDF格式的上市公司财务报告文件放入此文件夹
2. 运行分析程序：`python src/main.py`
3. 系统会自动识别并处理所有PDF文件

## 文件命名建议

为了便于识别和管理，建议使用以下命名格式：

```
公司名称_年度_报告类型.pdf
```

示例：
- `贵州茅台_2023_年度报告.pdf`
- `中国平安_2023_年度报告.pdf`
- `腾讯控股_2023_年度报告.pdf`

## 支持的文件格式

- PDF格式（.pdf）
- 文件大小：建议不超过100MB
- 文件编码：UTF-8 或 GBK

## 注意事项

1. 确保PDF文件完整且未损坏
2. 建议使用标准格式的财报PDF（包含完整的三大财务报表）
3. 系统会自动跳过非PDF文件
4. 处理完成后，Markdown文件会保存到 `repository/` 文件夹
5. 分析报告会保存到 `reports/` 文件夹

## 示例文件

您可以在此文件夹中放置示例PDF文件进行测试。系统支持批量处理多个文件。
