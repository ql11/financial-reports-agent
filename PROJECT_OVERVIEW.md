# 财报造假分析系统 - 项目概述

## 📋 项目简介

本项目是一个专业的财报造假分析系统，专门用于深度分析上市公司财务报告，识别财务异常和造假风险。系统基于真实财报数据，采用先进的财务分析算法，能够准确检测财报造假迹象。

## 🎯 项目目标

### 主要目标
1. **准确识别财报造假**：通过多维度分析，准确识别财报中的异常和造假迹象
2. **量化风险评估**：对财报造假风险进行量化评估，提供风险等级
3. **生成专业报告**：生成详细的分析报告，包括风险点、造假路径和建议措施
4. **支持投资决策**：为投资者提供基于风险的投资建议

### 技术目标
1. **基于真实数据**：使用真实财报数据进行深度分析
2. **多维度分析**：从多个维度分析财务数据
3. **智能检测**：自动识别财报造假迹象
4. **可扩展架构**：支持添加新的检测指标和分析维度

## 🔍 项目背景

### 问题发现
在分析英洛华2025年报时，发现原系统存在严重缺陷：
- 使用模拟数据，无法发现真实财报中的异常
- 只进行静态比率分析，缺乏趋势和逻辑分析
- 风险评级严重错误（原系统：低风险4.8/10，实际：极高风险）

### 改进需求
基于以上问题，需要对系统进行彻底改进：
1. **数据源改进**：从模拟数据改为真实财报数据
2. **分析方法改进**：增加趋势分析、逻辑关系分析
3. **检测指标扩展**：从7个指标扩展到17个关键指标
4. **风险模型改进**：重新设计风险评分模型

### 分析流程
1. **数据准备**：读取PDF财报文件，提取财务数据
2. **指标计算**：计算核心财务指标和异常指标
3. **异常检测**：检测财报造假和财务异常
4. **风险评估**：评估造假风险和财务风险
5. **报告生成**：生成详细的分析报告
6. **结果输出**：输出Markdown和JSON格式报告

## 📊 检测指标

### 1. 核心业绩背离指标（7个）
- 收入与利润背离
- 现金流与利润背离
- 收入与现金流背离
- 扣非净利润与归母净利润背离
- 政府补助依赖度
- 非经常性损益占比
- 递延收益摊销异常

### 2. 资产负债表异常指标（6个）
- 应收账款异常增长
- 坏账准备反向变动
- 存货异常增长
- 存货跌价准备反向变动
- 应收账款/收入比率
- 存货/收入比率

### 3. 运营效率指标（4个）
- 产能利用率趋势
- 在建工程异常增长
- 库存与销售增速差异
- 生产与销售增速差异

### 4. 公司治理指标（2个）
- 历史违规记录
- 关联交易风险

## 📈 案例分析

### 英洛华2025年报分析

#### 原系统分析结果（错误）
- **风险评级**：低风险（4.8/10，B级）
- **检测指标**：2个（关联交易异常、会计政策变更）
- **投资建议**：可考虑投资

#### 增强版系统分析结果（正确）
- **风险评级**：极高风险（45/50）
- **检测指标**：17个（14个高风险，3个中风险）
- **投资建议**：强烈建议暂停投资，进行深入尽职调查

#### 关键发现
1. **业绩严重背离**：收入下降3.11%，利润增长0.93%，现金流暴跌41.5%
2. **应收账款异常**：增长12.58%，坏账准备减少40万
3. **存货异常**：增长21.33%，跌价准备减少147万
4. **政府补助操纵**：增长115%，递延收益摊销增长26倍
5. **产能利用率下降**：固定资产周转率持续下滑，但在建工程增长54%
6. **历史违规前科**：2021年被浙江证监局警示

## 🔧 技术实现

### 核心技术
1. **PDF解析**：使用pdfplumber库提取PDF文本
2. **数据分析**：使用pandas和numpy进行财务分析
3. **异常检测**：基于阈值和逻辑规则的异常检测
4. **风险评估**：基于权重的风险评分模型
5. **报告生成**：基于模板的报告生成系统

### 关键算法
1. **业绩背离检测算法**：检测收入、利润、现金流的异常关系
2. **资产异常检测算法**：检测应收账款和存货的异常变动
3. **利润操纵检测算法**：检测非经常性损益对利润的影响
4. **运营异常检测算法**：检测产能利用率和投资效率异常
5. **风险评分算法**：基于多指标的风险评分模型

### 性能优化
1. **并行处理**：支持多文件并行分析
2. **缓存机制**：缓存分析结果，提高性能
3. **内存优化**：优化内存使用，支持大文件分析
4. **错误处理**：完善的错误处理和恢复机制
